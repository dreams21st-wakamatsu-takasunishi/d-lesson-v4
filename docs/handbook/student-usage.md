# 児童向け利用手引書

## 児童が使うもの

- DレッスンのURLまたはデスクトップのショートカット。
- 児童番号。
- あいことば。

児童には、メールアドレスやSupabase Auth User IDを伝えません。

## ログインする

1. Dレッスンを開く。
2. 児童番号を入力する。
3. あいことばを入力する。
4. `ログイン` を押す。
5. 自分の練習メニューが開く。

ログイン後に表示される名前が自分と違う場合は、練習を始めず先生に伝えます。

## 練習を始める

練習メニューから選びます。

- マウス練習
- キーボード練習
- 文章練習
- ことば入力
- ビジョントレーニング
- ガチャ
- 記録確認

## 練習後に保存されるもの

児童本人のアカウントで練習した場合、次の情報が保存されます。

- コイン
- クリアしたステージ
- マウスLv
- キーボード進捗
- ビジョン記録
- 文章練習記録
- ことば入力記録
- 取り組み履歴
- 前回取り組んだ練習

## 終わるとき

1. ホームまたはタイトル画面に戻る。
2. `ログアウト` を押す。
3. ログイン画面に戻ったことを確認する。

共有端末では、ログアウトせずに席を離れないでください。

## うまくいかないとき

### ログインできない

- 児童番号が正しいか確認する。
- あいことばが正しいか確認する。
- 何度も失敗する場合は先生に伝える。

### 自分の名前が出ない

- 練習を始めず先生に伝える。
- 管理者側で `lesson_user_access` の紐づけを確認する。

### 練習したのに進捗が変わらない

- 先生確認モードで開いていないか確認する。
- 通信状態を確認する。
- 画面を再読み込みして、進捗が残っているか見る。
