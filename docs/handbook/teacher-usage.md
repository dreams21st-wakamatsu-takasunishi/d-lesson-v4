# 先生向け利用手引書

## 先生アカウントの目的

先生アカウントは、児童の状態を確認するためのアカウントです。  
基本的には閲覧・確認用であり、児童データを書き換えない運用にします。

## ログインする

1. Dレッスンを開く。
2. 先生・管理者ログインを開く。
3. 先生用メールアドレスとパスワードを入力する。
4. ログインする。
5. タイトル画面に `先生用プレビュー` が表示されることを確認する。

## 見える児童

先生アカウントは、管理者が設定した範囲だけ表示できます。

- 全児童
- 指定グループのみ

グループ指定は `lesson_user_access.scope_type` と `scope_value` により制御されます。  
管理者画面の `Auth連携` で設定します。

## 先生用プレビュー

先生用プレビューでは、各練習画面の確認ができます。  
ただし、先生アカウントでの操作は児童の進捗保存に使わない設計です。

確認すべきこと:

- 先生確認モード表示が出る。
- 児童のコインや進捗が、先生操作で保存変更されない。
- 再読み込み後も、先生操作によるコインや進捗の変化が残っていない。

## 児童の状態確認

先生が確認する主な項目です。

- コイン
- マウスLv
- キーボード進捗
- 直近の取り組み
- 前回取り組んだ練習
- 管理者が出力した児童レポート

## 先生が行わない操作

次の操作は管理者が行います。

- 児童追加
- 児童削除
- 表示名変更
- 生年月日変更
- グループ変更
- Auth連携
- バックアップ復元
- 進捗編集
- 全体設定変更

## 先生アカウントの確認手順

1. 先生でログインする。
2. 担当グループの児童だけが見えることを確認する。
3. 指定外グループが見えないことを確認する。
4. 児童を1人選ぶ。
5. ステージを確認する。
6. 画面を再読み込みする。
7. コイン・進捗が先生操作で変わっていないことを確認する。
8. Consoleにアプリ本体由来のエラーがないことを確認する。
