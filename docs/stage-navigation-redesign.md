# ステージ導線見直し方針

## 目的

児童がログイン後に、今の進捗と次に取り組む内容をすぐ判断できるようにする。

## 現行の主な入口

- ホーム: 進捗、つぎのおすすめ、今日のミッション
- れんしゅう: マウス、キーボード、文章入力、ゲーム、ビジョン、Word
- キーボード: カテゴリ、章、ステージ
- ビジョン: 分野、ステージ、難易度
- マイページ: 記録、賞状、グラフ

## 推奨階層

```text
ホーム
  現在地
  次の候補
  今日のミッション
  れんしゅう一覧

れんしゅうカテゴリ
  コース
    章
      ステージ
        難易度
```

## コース分類

- 基礎操作: マウス、ABC導入
- 文字入力: キーボード基本、タッチタイピング、ひらがな、ことば
- 実用入力: 文章入力、Word
- 見る力: ビジョントレーニング
- 応用・自由: タイピングゲーム、リズムゲーム、自由時間

## 実装済みの第一段階

- ホームに `現在地` を追加。
- ホームに `次の候補` を最大3件表示。
- 大きい `つぎのおすすめ` と候補リストを同じ判定ロジックから表示。

## 実装済みの第二段階

- れんしゅうカテゴリ画面のカードに `コース分類` を追加。
- 各カードに `次のステージ` または `未完了状態` を表示。
- キーボードカテゴリカードは、ABC導入とキーボード本編を合算した進捗を表示。
- 文章入力、ビジョン、Wordは既存の保存データから最初の未完了課題・ステージを表示。
- Wordが未解放の場合は `ローマ字テスト後` と表示。

## 実装済みの第三段階

- 文章入力、ビジョントレーニング、Wordの各メニューに `つぎ` パネルを追加。
- `つぎ` パネルに、次に押す課題・ステージ、進捗、前回の取り組みを表示。
- ビジョントレーニングは、選択中の分野内で次のステージと難易度を表示。
- Wordは、次に進める章を表示し、該当カードを強調。
- 文章入力は、未完了課題の先頭を表示し、該当カードを強調。

## 次の段階

- 管理者設定で「標準ルート」と「教室ごとのおすすめ順」を切り替えられるようにする。
- 先生・管理者画面で、児童ごとの標準ルート到達状況を一覧確認できるようにする。
