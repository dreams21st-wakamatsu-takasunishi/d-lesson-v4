# 実操作時の未定義エラー確認手順

## 目的

ビルドでは検出できない `ReferenceError: xxx is not defined` や、画面操作時だけ発生する未定義エラーを見つける。

## 確認前の準備

1. 開発サーバーを起動する。
   - 例: `npm.cmd run dev -- --host 127.0.0.1`
2. ブラウザで対象URLを開く。
   - 例: `http://127.0.0.1:5174/`
3. 開発者ツールを開き、Consoleを表示する。
   - Windows: `F12` または `Ctrl + Shift + I`
4. Consoleを一度クリアする。
5. ページを再読み込みする。
   - 可能なら `Ctrl + F5`

## 共通の確認ルール

- 画面を開いた直後にConsoleへ赤いエラーが出ていないか確認する。
- ボタン、カード、戻る、ホーム、ログアウト、閉じる、キャンセル、決定を押した直後に確認する。
- `ReferenceError`、`is not defined`、`Cannot access ... before initialization` は必ず報告する。
- `TypeError` も未定義に近い原因の可能性があるため報告する。
- `/favicon.ico 404` だけの場合は、今回の未定義エラー対象外として扱う。

## 確認すべき画面と操作

### 1. 初期画面・ログイン

- 初期表示
- 学年選択
- 児童選択
- 生徒ログイン
- 先生ログイン
- ログアウト
- ホームボタン
- 戻るボタン
- BGM、効果音、全画面切り替え

### 2. マウス練習

- マウスメニューを開く
- 各ステージボタンを押す
- 練習開始
- 正解操作
- 不正解操作
- クリア画面
- もう一度
- メニューへ戻る

### 3. キーボード練習

- キーボードカテゴリを開く
- 基本、ブラインド、ブリッジをそれぞれ開く
- チャプターを開く
- ステージを開く
- 練習開始
- 正解入力
- 不正解入力
- リトライ
- メニューへ戻る
- 苦手練習を開く

### 4. テキスト練習

- テキストメニューを開く
- 課題を開く
- 開始確認を表示する
- キャンセルする
- もう一度開始する
- ルビ切り替え
- ナビ切り替え
- 提出する
- 結果画面を閉じる
- メニューへ戻る

### 5. ことば入力

- ことばメニューを開く
- ステージカードを押す
- 課題を開く
- 中断する
- 再開する
- 先生確認を開く
- キャンセルする
- メニューへ戻る

### 6. ビジョントレーニング

- ビジョンメニューを開く
- 各ステージを開く
- 選択肢を押す
- タイマー終了まで待つ
- クリア、失敗、再挑戦を確認する
- メニューへ戻る

### 7. ガチャ・チケット・着せ替え

- マイページまたは記録画面を開く
- ガチャを開く
- 1回ガチャを押す
- 確認ダイアログでキャンセルする
- もう一度ガチャを押して実行する
- 報酬画面を閉じる
- チケット使用
- テーマ変更
- 演出変更

### 8. 記録・管理者画面

- 記録画面を開く
- 各タブを切り替える
- CSV出力ボタンを押す
- 管理者画面を開く
- 児童追加
- コイン追加
- チケット設定保存
- 進捗編集を開く
- 進捗編集を保存またはキャンセルする
- ダッシュボードのタブ切り替え
- カスタムテーマ作成を開く
- カスタム演出作成を開く

## 報告すべき項目

エラーが出た場合は、次をセットで報告する。

```text
発生日時:
URL:
操作したユーザー種別: 生徒 / 先生
児童名またはテストユーザー名:
直前にいた画面:
押したボタンまたは操作:
再現手順:
Consoleのエラー全文:
エラーに表示されたファイル名と行番号:
画面上の見た目:
期待していた動き:
毎回起きるか: 毎回 / ときどき / 1回だけ
スクリーンショット: あり / なし
```

## 報告例

```text
発生日時: 2026-04-30 15:20
URL: http://127.0.0.1:5174/
操作したユーザー種別: 生徒
児童名またはテストユーザー名: テスト太郎
直前にいた画面: ことばメニュー
押したボタンまたは操作: 最初のステージカードをクリック
再現手順:
1. テスト太郎でログイン
2. ことばメニューを開く
3. 最初のステージカードをクリック
Consoleのエラー全文:
ReferenceError: currentWordStageId is not defined
    at startWordStage (...)
エラーに表示されたファイル名と行番号: index-xxxxx.js:28853
画面上の見た目: ステージが開かず、画面が止まった
期待していた動き: ことば課題画面が開く
毎回起きるか: 毎回
スクリーンショット: あり
```

## 優先度

- 高: 生徒ログイン、先生ログイン、マウス、キーボード、テキスト、ことば、ガチャで発生するエラー
- 中: 記録、ダッシュボード、カスタムテーマ、カスタム演出で発生するエラー
- 低: 表示崩れのみ、または `/favicon.ico 404` のみ
